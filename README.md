# LZW-py
Lempel–Ziv–Welch (LZW) algorithm implementation in pure Python.


usage: lzw.py [-h] -i I -o O [-a] [-d | -c]

optional arguments:
  -h, --help  show this help message and exit
  -i I        Input file path
  -o O        Output file path
  -a          Proceed all files in input directory
  -d          Decompress
  -c          Compress
